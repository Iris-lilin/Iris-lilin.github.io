彩云之南--丽江篇

发表于 2017-03-22 | 分类于 随笔

     # 彩云之南 #
  呼吸了西安六个月的雾霾后，我们选择期末考完去云南洗洗肺，当天落地丽江的时候已经凌晨1点多了，对于一个在学校11点前入睡的人来说简直相当于锥刺股啊，来不及多呼吸几口就立马入酒店休息了。

  • day one

  听说8点前进古城不需要维护费，我们都订了7点的闹钟，嗯~意料之中，几个人你等我起我等你收拾，出酒店也差不多9点半了。走在丽江的街道，感觉……人不多，太阳很刺眼，天特别蓝，没有云，远处的雪山看的特别清楚，随处看得到用背篓背小孩的妇女。
在两个云南当地小伙伴的带领下，我们的早餐就是当地有名的饵丝，炒饵丝，扒肉饵丝，焖肉饵丝……嗯~ 第一次吃感觉太好吃了，故：无照片！

  • 第一站：束河古镇

  刚进古镇就被老奶奶拉着辫五彩辫，好像本来是10元4个，反正讲价之后是10元12个（在古城除非明码标价，一定要讲价！）还有古城转角拍的马


      



  一个很美的拱桥，好多人在拍照，我俩也凑热闹，哈哈哈 右图放大后谜一般的画质

      


  和小伙伴在古镇走丢后，小伙伴自己玩的好像挺(｡･∀･)ﾉﾞ嗨，拿他的照片用一用~

      

  • 第二站：拉市海

  某宝上78元吃土鸡，骑马，划船全套强烈推荐！图一是吃土鸡的地方，在我们前面还有几波人。吃完土鸡开始骑马，路上还有很多返程的旅游团


      


  第一次骑马，挺害怕的（还有一小伙伴由于某种原因，压根没上马）特别是山路特别陡，刚开始上山马踏过去尘土飞扬，幸亏小伙伴借衣服给我穿~感谢山顶fashion的老奶奶教我们怎么摆pose，怎么跳，约定下次来丽江找她玩

      



      


  骑马回来就去划船，天蓝海清，划不动漂着很惬意，起风了，感觉要上天

      



      

  • 第三站：丽江古城

  玩完后土鸡师傅直接把我们送到了一家火锅店门口，吃了有名的黑山羊，腊排骨火锅，水性杨花原来是这样，第一次见到血淋林的羊血，纯液体啊（sorry，吃饭不喜欢拍照~）吃饱喝足出发去古城，从一条很窄的路一直上坡，走完别有洞天就是古城，半路上一度怀疑给我们指路的人瞎指的，哈哈哈~

  说好的古城艳遇呢？没人搭讪！！！

  古城里的一个蜡像馆，小女孩真可爱（没说我寄己），在一家轻吧里俯瞰古城夜景，酒吧N条街啊



      

  小伙伴买了一个看起来很高大上的纸书灯（名字我自己描述的），是用那个什么文写的（工科女此时哭成狗），我们买了。。。吃的。。。。哈哈哈~

###第一天打卡 滴~
